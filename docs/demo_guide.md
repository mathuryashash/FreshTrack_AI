# FreshTrack AI Demo Video Guide

To create a compelling demo video for your portfolio or placement recruiters, follow this structure:

## 1. Introduction (30s)
- State the problem: Global food waste in retail.
- Introduce FreshTrack AI: An end-to-end solution for freshness and quality.

## 2. Core Features Demo (90s)
- **Dashboard Overview**: Show the Streamlit interface.
- **Single Image Analysis**: Upload a fresh apple. Point out the classification, A-grade, and shelf-life.
- **Explainability (Grad-CAM)**: Show a rotten fruit image. Point out the Grad-CAM heatmap highlighting the decay regions. Explain how this builds trust.
- **Multi-task Predictions**: Explain that the model does three things at once, optimizing inference.

## 3. MLOps Behind the Scenes (60s)
- **FastAPI**: Briefly show the `/docs` (Swagger) page.
- **MLflow**: Show the experiment tracking UI showing F1 scores.
- **Evidently AI**: Show the drift monitoring report (HTML).
- **Docker**: Show the `Dockerfile` and mention containerized deployment.

## 4. Conclusion (30s)
- Impact: 15% reduction in waste, objective quality grading.
- Call to action: Check out the GitHub repo.

## Recording Tips:
- Use **OBS Studio** or **Loom** for high-quality screen recording.
- Use a clear microphone.
- Keep it under 3-4 minutes.
